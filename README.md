## ğŸ“¦ Missing Assets / Not Included in This Repository

Some files are excluded intentionally (for repo size, reproducibility, and security):

### 1) Large binary assets
Examples:
- Large 3D models (e.g., `*.glb`)
- Large videos (`*.mp4`, `*.mov`)
- Large textures/audio files

**If a scene requires a missing large model**, you must provide the asset yourself and place it into the correct `Assets/` folder path (keeping the original folder structure).

### 2) Generated Unity files (regeneratable)
Examples:
- Baked lighting outputs such as `LightingData.asset`

These are generated by Unity and may differ across machines. They are not required in version control.

### 3) Build artifacts (not versioned)
Examples:
- `*.apk`, `*.aab`

### 4) Sensitive signing materials (never commit)
Examples:
- `*.keystore`, `*.jks`

> âš ï¸ **Signing keys and passwords are NOT included in this repository.**  
> For release builds, you must create and configure your own keystore.
